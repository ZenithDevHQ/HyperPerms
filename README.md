# HyperPerms

A powerful permissions management plugin for Hytale servers.

## Download

Get the latest version from [Releases](https://github.com/ZenithDevHQ/HyperPerms/releases).

## Features

- Advanced permission management with groups and inheritance
- Context-aware permissions (world, gamemode, time, biome, region)
- Web editor for easy configuration
- Chat formatting with prefix/suffix support
- Automatic backup system
- Auto-update notifications

## Installation

1. Download the latest JAR from releases
2. Place in your server's `mods/` folder
3. Restart the server

## Commands

| Command | Description |
|---------|-------------|
| `/hp help` | Show all commands |
| `/hp version` | Check version and updates |
| `/hp reload` | Reload configuration |
| `/hp group` | Manage groups |
| `/hp user` | Manage users |
| `/hp editor` | Open web editor |
